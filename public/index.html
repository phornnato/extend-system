Extend System API Documentation

All API requests should be prefixed with /api. Timezone is set to Asia/Phnom_Penh.

Categories API

GET /api/categories

List all categories.

ParameterTypeRequiredDescription
typestringNoFilter by income or expense
POST /api/categories

Create a new category.

KeyTypeRequiredDefault / Options
namestringYesCategory name
typestringYesincome, expense

Incomes API

GET /api/incomes

Fetch all income records with categories.

POST /api/incomes

Add a new income record.

KeyTypeRequiredDescription
amountnumericYesThe decimal amount
descriptionstringYesShort note
datedateYesYYYY-MM-DD
category_idintegerNoID from Categories table
PUT /api/incomes/{id}

Update existing income record (all keys optional in request body).
